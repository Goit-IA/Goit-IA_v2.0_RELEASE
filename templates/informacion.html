Ingeniería detrás de Goit-IA

Un sistema de Razonamiento Semántico construido con arquitectura RAG.

01

Búsqueda (KNN)

Un modelo K-Nearest Neighbors realiza una búsqueda semántica para clasificar y encontrar la información oficial más relevante.

02

Generación (LLM)

Un Modelo de Lenguaje Grande procesa los datos recuperados para redactar una respuesta natural y coherente.

Sistema RAG

Recuperación Aumentada

La unión crea una respuesta perfecta: primero recuperamos datos precisos de los trámites y luego generamos la explicación ideal para ti.
